실력키우기 > 수학 > 이진수 (2814) 문제 클릭

이번 문제는 2진수를 입력받아 10진수로 변환하난 간단한 문제입니다. 일단 숫자로 받을 경우 10진수로 인식하기 때문에, 2진수를 입력받으려면 문자로 받아야 합니다.

문자로 받고 나서 10진수로 변환해야 하는데 방법은 2가지 입니다. 일단 10진수의 각 자리가 10의 0승(1), 10의 1승(10), 10의 2승(100), … 이런 식으로 되어있는 것처럼, 2진수도 마찬가지입니다. 뒤에서부터 2의 0승(1), 2의 1승(2), 2의 2승(4), 2의 3승(8), … 이런 식으로 되어있기 때문에 각 자리수에 접근해서 자리수를 곱해준 다음에 더하면 계산할 수 있습니다.

여기서 String 객체의 각 문자에 접근할 때는 charAt 이라는 메소드를 이용합니다. 또한 2의 n 승을 구할 때는 Math.pow() 라는 메소드를 이용합니다. return 값이 double 타입이기 때문에 int 로 형변환도 해줍니다.

하지만 더 쉬운 방법이 있습니다. Integer.parseInt() 라는 메소드는 입력받은 문자열을 숫자로 변환해주는 메소드로 굉장히 많이 쓰이는 메소드입니다. 그런데 이 메소드에 기수 (radix)를 뜻하는 int 타입 두번째 매개변수를 넣을 수 있습니다. 사용법은 다음과 같습니다.
