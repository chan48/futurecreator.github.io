정올(JUNGOL) 전화번호 속의 암호(1620) 알고리즘 기초 문제 풀이

2017-01-19
  • algorithm
  • basic

실력키우기 > 문자열 > 전화번호 속의 암호 (1620) 문제 클릭

점점 문제가 길어지고 있네요. 지문은 길지만 문제는 굉장히 간단합니다. ‘-‘ 로 구분된 숫자가 연속된 문자열을 입력받아서 특정 더미를 선택하고, 각 자리수마다 특정 숫자를 더해서 출력하면 됩니다.

각 숫자 더미는 4자리로 정해져있지만, 4자리보다 적게 들어올 수도 있고 많이 들어올 수도 있습니다. 적은 경우는 부족한만큼 0을 붙여서 4자리를 만들어야 하고, 많은 경우는 에러 메시지를 출력해야합니다.

0을 붙여서 4자리를 만드는 경우를 생각해봅시다. Oracle SQL 처럼 LPAD 함수가 있으면 좋겠지만 Java 에는 없네요. 원리는 이렇습니다. 들어오는 자리가 1 ~ 3자리인 경우에 왼쪽에 0을 붙여야하는 상황이므로 앞쪽에 먼저 0을 4개 붙인후 뒤쪽에서부터 4개만큼 자르면 됩니다. 뒤쪽에서부터 4개는 어떻게 자르면 될까요? substring() 메소드를 활용하면 되는데, 해당 문자열의 길이에서 4만큼 뺀 index 부터 잘라내면 되겠죠?

추가로 말씀드리면, substring() 메소드는 파라미터를 하나만 받을 수도 있고, 두 개를 받을 수도 있습니다. 이러한 성질을 오버로딩(Overloading)이라고 하죠. 파라미터를 하나만 받는 경우는 해당 index 부터 마지막까지 글자를 잘라냅니다. 두 개를 받는 경우는 시작점과 끝점을 의미하는데, 이게 끝점의 경우 실제적으로는 그보다 한 자리 앞입니다. 이게 익숙하지 않은 경우 헷갈리기 쉬운데 이 때는 첫번째 파라미터가 시작점을 의미하고, 두번째 파라미터 - 첫번째 파라미터가 잘라내는 문자의 길이라고 생각하시면 쉽습니다. 하단의 코드에서 substring(2, 5) 가 2부터 시작해서 3개의 문자를 잘라낸다고 생각하시면 됩니다 (5 - 2 = 3개).

처음 부분에서 String[] 배열을 만들 때 split() 메소드를 사용했습니다. 매개변수로 “-“ 를 넘겨주면 “-“ 를 기준으로 해서 문자를 잘라내 String[] 을 만들어줍니다.
