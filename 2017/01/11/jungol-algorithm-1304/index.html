정올(JUNGOL) 숫자사각형3(1304) 알고리즘 기초 문제 풀이

2017-01-11
  • algorithm
  • basic

실력키우기 > 도형만들기 > 숫자사각형3 (1304)

정사각형 한 변의 길이 n 을 입력받아 n x n 형태로 숫자를 출력하는 문제입니다. 출력 순서는 가로 방향이 아니라 세로 방향으로 출력합니다. 좀 더 쉽게 접근할 수 있는 방법은 이차원 배열을 선언 후에 세로 방향으로 숫자를 채워나간 후에 출력하면 됩니다.

저는 배열 없이 바로 출력하기 위해서 각 열에 사이즈 n 을 곱해서 숫자를 계산했습니다. 어렵지 않게 풀 수 있는 문제입니다.
