정올(JUNGOL) 2진수를 10진수로...(1274) 알고리즘 기초 문제 풀이

2017-01-18
  • algorithm
  • basic

실력키우기 > 수학 > 2진수를 10진수로… (1274) 문제 클릭

저번 문제와 마찬가지로 2진수를 10진수로 바꾸는 문제입니다. 다만 이번에는 최상위비트가 부호비트인 점이 다르네요. 입력은 8비트 2진수이고, 최상위비트는 값이 아닌 부호를 의미합니다. 0이면 양수, 1이면 음수입니다.

이 부호가 양수냐 음수냐에 따라서 로직이 달라집니다. 음수인 경우 부호 비트를 제외하고 나머지 부분을 10진수로 변환한 후에 마이너스 부호(“-“) 만 붙이면 될 것 같지만 전혀 그렇지 않습니다. 이 경우에는, 각 자리수의 비트를 0 -> 1, 1 -> 0 으로 반전시켜 주고나서 마지막 비트에 1을 더해서 새로 계산한 후, 10진수로 변환해야 정확한 값으로 변환됩니다. 각 자리수의 비트를 역전시키는 것을 1의 보수(1’s Complement) 라고 하고 1의 보수의 마지막 비트에 1을 더한 것을 2의 보수(2’s Complement) 라고 합니다.

이제 구현만 하면 되겠네요. 최상위비트를 확인해서 양수인 경우에는 그냥 Integer.parseInt() 메소드를 이용해 바로 2진수를 10진수로 변환하면 되겠습니다. 음수인 경우 2의 보수를 구해야하는데, 각 자리수의 계산을 좀 더 쉽게 하기 위해 String 으로 받은 값을 int 타입 배열을 이용해 저장했습니다.
