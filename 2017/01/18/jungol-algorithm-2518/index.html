정올(JUNGOL) 문자열변환(2518) 알고리즘 기초 문제 풀이

2017-01-18
  • algorithm
  • basic

실력키우기 > 문자열 > 문자열변환 (2518) 문제 클릭

입력받은 교정 정보를 이용해서 해당 문자열을 교정하는 문제입니다. 문제 자체는 크게 어렵지 않습니다.

저는 교정 전후 문자 정보를 함께 저장하기 위해서 클래스를 하나 만들었습니다. 물론 그냥 char[] 배열 여러개를 선언해서 사용하셔도 상관없습니다. 오타가 포함된 문자는 문자열이 아닌 문자 형태로 하나씩 입력되고 있네요. 이걸 char 타입으로 받고 싶으시다면, Scanner 자체에는 nextChar() 이런 메소드가 없습니다. 따라서 next().charAt(0) 으로 문자열로 받으신 후에 첫번째 문자에 접근해서 가져오시는 방법이 있습니다.

그리고 한가지 주의하실 점이 있습니다. 앞서 교정한 문자가 뒤에서 교정할 때 또 대상이 되어서 여러번 교정이 될 수 있습니다. 예를 들어, 기존 문자열에서 a 를 b 로 바꾸고, b 가 있으면 c 로 바꿔야 한다고 했을 때, 원본에서의 a 는 b 로 바뀌어야지 두번 바뀌어서 c 로 바뀌면 안됩니다.

이렇게 하기 위해서 이미 교정이 된 문자는 교정이 되었다고 표시를 해야 합니다. 저는 int[] 배열을 이용해서 1로 체크를 해줬습니다.
